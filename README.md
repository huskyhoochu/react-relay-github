# React-Relay-Github

## Summary

react와 relay를 이용해 github graphql api를 호출하는 프로젝트입니다.

## Usage

```
yarn install // 의존성 패키지 설치

yarn build // 프로덕션 빌드

yarn start // 프로덕션 실행

yarn dev // 개발 환경
```

## .env

```
GITHUB_TOKEN="자신의 personal access token (repo 및 email 권한 필요)"
LOGIN_NAME="검색할 Github 로그인 이름"
```
